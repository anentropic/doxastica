# Tutorials

New to doxastica? Start here. These lessons take you by the hand and guarantee a working result: you learn the library by building with it, with every term defined as you meet it.

- **[Your First Belief Store: Revise, Supersede, Time-Travel](first-belief-store.md)**: Build a core from scratch and exercise every operation: revise, expand, contract, walk the revision chain, and reconstruct a past state.
- **[Build a Shared-World Memory for an Agent Team](agent-team-memory.md)**: A multi-scope use case. Model each agent as a scope, keep shared truth in the world scope, and use get_impact to find which agents a change to that truth makes stale.

When you have finished the tutorial and want to solve specific problems, move on to the [How-To Guides](../how-to/index.md). To understand the ideas behind what you built, read the [Explanation](../explanation/index.md) section.
