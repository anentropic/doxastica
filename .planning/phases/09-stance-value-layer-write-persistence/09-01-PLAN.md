ladybug schema — so a stance flows write -> persist -> read on BOTH backends with `certain` as
the API default. This is the atomic production change of Phase 9.

(D-01), the model edit, the `core.py` threading, the Protocol signatures, and the ladybug DDL

or a props dict targeting a missing column). This plan lands all of it plus the type-level
unit proof.
