<objective>
Prove — mechanically, on BOTH backends — that stance survives the full write -> persist -> read
loop byte-stable: it round-trips through `query_scope`, defaults to `certain` when omitted, is
preserved verbatim by `contract` on the retracted tail, and is reconstructed by `get_scope_at`.

Purpose: This is the headline guarantee of Phase 9 (ROADMAP SC4/SC5). Plan 09-01 landed all the
production plumbing and proved nothing regressed; it did NOT assert stance is actually *preserved*
(the existing suite predates stance). This plan supplies that proof, and does so across the
`memory` + `ladybug` parity fixture so byte-stability is proven at the core boundary on both
substrates.

Output: `tests/test_stance_persistence.py` — four parametrized parity tests (STANCE-03/04/05).
</objective>
