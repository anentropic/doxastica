    - upsert_node is idempotent: upserting the same node_id twice yields one node; SET-style props update on re-upsert.

    - match_nodes returns nodes of a label whose props AND-exact-match the where map; empty where returns all of that label.

    - traverse with finite max_depth returns nodes within depth and a frontier = node-ids at exactly max_depth that have an unexpanded successor.

    - BackendDependencyError is catchable as both DoxasticaError and ImportError.
